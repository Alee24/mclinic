# UI/UX Design System & Concepts

## 1. Design Philosophy
- **Clean & Trustworthy**: Medical apps must convey trust. Use ample whitespace, clear typography, and simplified navigation.
- **Accessibility**: High contrast for elderly users. Screen reader compatibility.
- **Responsive**: Mobile-first design.

## 2. Color Palette (Logo Derived)
- **Primary Brand**: `#00F090` (Bright Green - *M-Clinic Text*) - Energy, Growth, Medical safely.
- **Heart Red**: `#E91E63` (Pinkish Red - *Heart Icon*) - Vitality, Care.
- **Pulse Blue**: `#B3E5FC` (Light Blue - *Heartbeat Line*) - Calm, Technology.
- **Dark**: `#1A1A1A` (Text)
- **Light**: `#FFFFFF` (Backgrounds)

## 3. Typography
- **Font Family**: *Inter* or *SF Pro Display*.
- **Headings**: Bold, Clear hierarchy (H1 for page titles, H2 for sections).
- **Body**: 16px min size for readability.

## 4. Key Screens & Concepts

### 4.1 Patient Dashboard
*   **Header**: Greeting ("Good Morning, John"), Notifications icon.
*   **Quick Actions**: Large cards for "Book Appointment", "Order Ambulance", "Symptom Checker".
*   **Health Summary**: Widget showing recent vitals (Heart rate from wearable) or upcoming appointments.
*   **Bottom Nav**: Home, Calendar, Records, Profile.

### 4.2 Booking Flow
1.  **Search**: Search bar for specialty/doctor name + Filters (Location, Rating).
2.  **Results List**: Doctor cards showing photo, specialty, years experience, next available slot.
3.  **Profile View**: Full bio, verified badges, reviews.
4.  **Slot Picker**: Calendar grid.
5.  **Payment**: M-Pesa phone number prompt.

### 4.3 Female Health Tracker
*   **Visuals**: Softer accent colors (Purple/Pink gradients).
*   **Cycle Wheel**: Interactive circular calendar for period tracking.
*   **Insights**: AI-driven tips based on cycle phase.
